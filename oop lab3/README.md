## ЛАБОРАТОРНАЯ РАБОТА № 03
### ТЕМА: НАСЛЕДОВАНИЕ, ПОЛИМОРФИЗМ
### Цель:
- Изучение механизмов работы с наследованием в С++;
- Изучение механизма перегрузки операци
### Вариант №34
Квадрат, прямоугольник, трапеция

### Запуск
```
mkdir build
cd build
cmake ..
cmake --build .
```
Вызов тестирования:
```
./tests
```
Вызов функции основной программы:
```
./main
```
